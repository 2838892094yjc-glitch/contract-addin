隐私政策

最后更新：2026年1月8日

概述

PEVC合同助手（以下简称"本加载项"）是一款用于辅助填写合同文档的 Microsoft Word 加载项。我们非常重视您的隐私，本政策说明我们如何处理您的数据。

数据收集

本加载项不会收集、存储或传输任何个人身份信息。

  • 所有表单数据仅存储在您的浏览器本地存储 (LocalStorage) 中
  • 文档内容仅在您的设备和 Microsoft Word 之间处理
  • 我们不运营任何后端服务器来收集数据

第三方服务

本加载项可能使用以下第三方服务：

  • Microsoft Graph API：用于 OneDrive 文件同步（需要您授权登录）
  • GitHub Pages：用于托管加载项界面

数据安全

所有数据传输均通过 HTTPS 加密。您的文档内容不会离开 Microsoft Office 环境。
